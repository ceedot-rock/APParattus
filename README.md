# APParattus

A Vercel-ready Next.js application for the APParattus product launch surface.

## Local development

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run start
```

The app uses Next.js App Router, TypeScript, responsive CSS, SEO metadata, robots, sitemap, and Node.js 22 for Vercel builds. Run `npm run typecheck` after installing dependencies to validate types locally.
